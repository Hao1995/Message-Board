# CRUD_Messages
Using Xampp and Laravel 5.2 to build a simple crud application.
Relational skills:
1. php
    - laravel 5.2
    - xampp
2. html
3. css
    - bootstrap
3. mysql

## Flow
``
> git clone git@github.com:Hao1995/Message-Board.git
> put this folder into your xampp/htdocs 
> add "crud_messages" schema in your "phpmyadmin"
> php artisan migrate
``
